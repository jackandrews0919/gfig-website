Control your GFIG Discord server — manage roles, channels, broadcast messages, and run the full server setup.

1. Go to discord.com/developers → Create application → Add a Bot → copy the token.
2. Deploy the discord-bot/ folder to Railway / Render (free). Set env vars: BOT_TOKEN · GUILD_ID · ADMIN_SECRET · PORT
3. Paste the deployed URL + your secret below. The bot will manage your server from this panel.
